Definition, Usage and Best Practices

Definition

The iframe tag specifies and inline frame. An inline frame is used to embed another document within the current HTML document.
(source: http://www.w3schools.com/tags/tag_iframe.asp)

Iframes were born out of the age of frame bassed layouts. When the internet was young and slow (Think 24k modems)Frame Layouts were used to isolate content in order to optimize loading when working with a mix of dynamic and static content. As the internet sped up, iframes outlived its frameset ancestors by continuing to be usefull means of embeding foreign content.

Iframes can be used for a number of Common applications of iframes include the embeding of multimedia, such as a YouTube video, into a site which dosn't host the content itself. Note how iframes can be displayed within iframes.

Note how YouTubes video player is used to present the video. Iframes enable for not only the borrowing of content but also the means of which it is displayed.

I frames enable us to display one website within another. An example of this application of iframes can be seen below.

The attributes of iframes, such as size, visibility, scrolling, in combination with styling allowed for the customization of displayed content. The width and height attributes of the following iframe are set to 1000x600 pixels while the src is set to the path of the content, in this case a pdf, being displayed. Since iframe is a relic of the early internet, mainy of its attributes can be defined by using either the HTLM within the tag or CSS with src being an exception.
